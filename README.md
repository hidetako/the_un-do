# 一息 — Issoku ｜ オンライン座禅寺「坐雲堂（The Un-do）」

24時間、世界中の誰もが自由に出入りできるインターナショナル座禅寺。
入口でひと息（一息）入れて、寺「坐雲堂」に入って座る。

- **サービス名**: 一息（Issoku） … `issoku.org`（取得予定）
- **寺名**: 坐雲堂（The Un-do） … The(ザ)＝坐／Un-do＝雲堂・undo・手放す
- **背骨の思想**: 手放す（只管打坐）。記録で競わせない。庭は「作務」として今日だけ。

## プロトタイプの流れ

一息（スプラッシュ：飛白の「調息」）→ 坐雲堂に入る → 姿勢（床/椅子）→ 座処6種
→ 時間スライダー（1〜60分）→ 調息オリエン（日英・姿勢別）→ 鐘3回 → 線香タイマー → 鐘1回 → 終わり

- 鐘は Web Audio で合成（低音・余韻長め）。日英切替・設定記憶・途中退室対応。

## ローカルで動かす

```bash
python -m http.server 5173
# http://localhost:5173 を開く（音は最初のクリック後に鳴る）
```
