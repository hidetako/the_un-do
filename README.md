# 一息 — Issoku ｜ オンライン座禅寺「坐雲堂（The Un-do）」

24時間、世界中の誰もが自由に出入りできるインターナショナル座禅寺。
入口でひと息（一息）入れて、寺「坐雲堂」に入って座る。

- **サービス名**: 一息（Issoku） … `issoku.org`（取得予定）
- **寺名**: 坐雲堂（The Un-do） … The(ザ)＝坐／Un-do＝雲堂・undo・手放す
- **背骨の思想**: 手放す（只管打坐）。記録で競わせない。庭は「作務」として今日だけ。

## プロトタイプの流れ

一息（スプラッシュ：飛白の「調息」）→ 坐雲堂に入る → 姿勢（床/椅子）→ 座処6種
→ 時間スライダー（1〜60分）→ 調息オリエン（日英・姿勢別）→ 鐘3回 → 線香タイマー → 鐘1回 → 終わり

- 鐘は Web Audio で合成（低音・余韻長め）。**座処ごとの環境音**（本堂のうなり／枯山水の風と鹿威し／海辺の波／夜の虫の音／猫の寝息／調息の額の静寂）も Web Audio で合成し、ヘッダーの **♪** ボタンで切替（設定記憶）。日英切替・途中退室対応。
- 坐雲堂（入口＝設定画面）に**光の地球儀**＝静かな共在感（§3.5）。**Supabase Realtime Presence で“いま堂にいる人”を実データ共有**し、その人の灯りが地球に灯る（坐っている人はより明るく）。「いま N の地域から M人がこの堂に」を日英でそっと表示。**位置はタイムゾーンからおおまかに・匿名／DBには何も書かない（揮発＝保存しない）**。接続できないときは数を偽らず「世界の坐雲堂は、いつでも開いています」に静かに退く。
